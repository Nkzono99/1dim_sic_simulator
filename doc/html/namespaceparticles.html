粒子を分配する.

分配方法

  • 'uniform' : startからendまでの範囲に一様に分配する
  • 'random' : startからendまでの範囲にランダムに分配する
  • 'normal' : 平均mean, 分散stdの正規分布に従うように分配する
Parameters
[in]npcl_add追加する粒子数
[in]ispec追加する粒子の種類
[in]x_mode位置の分配方法('uniform', 'random', 'normal')
[in]vx_mode速度の分配方法('uniform', 'random', 'normal')
[in]x_start初めの位置 (default: 0)
[in]x_end最後の位置 (default: ngrid*dx)
[in]x_mean平均位置 (default: 0.5*ngrid*dx)
[in]x_std位置の分散 (default: dx)
[in]vx_start最小速度 (default: -1)
[in]vx_end最大速度 (default: 1)
[in]vx_mean平均速度 (default: 0)
[in]vx_std速度の分散 (default: 1)
