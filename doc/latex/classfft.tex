\subsection{Detailed Description}
2の乗数長の配列に対する1次元\-F\-F\-T(Fast Fourier Transform)モジュール. 

\begin{DoxyAttention}{Attention}
fftを行う前にfft\-\_\-initを呼び出す必要あり.
\end{DoxyAttention}
使用例\-: 
\begin{DoxyCode}
\textcolor{keywordtype}{integer}, \textcolor{keywordtype}{parameter} :: n = 32
\textcolor{keywordtype}{real(8)} :: xs(n)  \textcolor{comment}{! x(t)}

call random\_number(xs(1:n))

call fft\_init(n)
fft\_buf(1:n) = xs(1:n)

\textcolor{comment}{! x(t) to x(k)}
call fft\_transform
xs(1:n) = fft\_buf(1:n)

\textcolor{comment}{! x(k) to x(t)}
call fft\_inverse\_transform
xs(1:n) = fft\_buf(1:n)
\end{DoxyCode}
