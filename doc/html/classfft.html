Detailed Description

2の乗数長の配列に対する1次元FFT(Fast Fourier Transform)モジュール.

Attention
fftを行う前にfft_initを呼び出す必要あり.

使用例:

integer, parameter :: n = 32
real(8) :: xs(n) ! x(t)
call random_number(xs(1:n))
call fft_init(n)
fft_buf(1:n) = xs(1:n)
! x(t) to x(k)
call fft_transform
xs(1:n) = fft_buf(1:n)
! x(k) to x(t)
call fft_inverse_transform
xs(1:n) = fft_buf(1:n)
